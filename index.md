## Brief Intro

Welcome to my cybersecurity portfolio.

I am a System Administrator building hands-on experience in cybersecurity engineering, cloud security, endpoint management, detection, and offensive security fundamentals. My background includes infrastructure administration, automation, networking, virtualization, Microsoft cloud administration, and security-focused troubleshooting.

This portfolio documents the labs, research, and technical projects I work on in my free time as I continue developing the skills needed to move into a dedicated Cybersecurity Engineer role.

My goal is to understand both sides of cybersecurity: how systems are attacked, and how they can be hardened, monitored, and defended.

## Featured Labs

### [T-Pot Honeypot Deployment](t-pot_deployment.md)

This lab documents my process of deploying a T-Pot honeypot inside a segmented network environment. The project focuses on learning more about exposed services, firewall rules, network segmentation, attack visibility, and security monitoring.

---

### [Privacy-Focused Pentest Workstation](untraceable_pentest_device.md)

This lab documents my process of building a privacy-focused pentest workstation using isolated virtual machines and controlled routing. The purpose of this project is to better understand traffic isolation, Tor-routed environments, and operational security concepts in an ethical lab setting.

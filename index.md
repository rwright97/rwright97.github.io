## Featured Labs

### [T-Pot Honeypot Deployment](t-pot_deployment.md)

This lab documents my process of deploying a T-Pot honeypot inside a segmented network environment. The project focuses on learning more about exposed services, firewall rules, network segmentation, attack visibility, and security monitoring.

---

### [Privacy-Focused Pentest Workstation](untraceable_pentest_device.md)

This lab documents my process of building a privacy-focused pentest workstation using isolated virtual machines and controlled routing. The purpose of this project is to better understand traffic isolation, Tor-routed environments, and operational security concepts in an ethical lab setting.

---

### [Sigma Detection Rule: Linux Privilege Escalation via Sudo Find](Sigma_Detection_Rules.md)

Simulated SSH initial access with Hydra, exploited a sudoers misconfiguration using <code>/usr/bin/find</code>, reviewed Linux authentication logs, and created a validated Sigma rule to detect the privilege escalation behavior.

---

### [Secure Web Gateway + Wazuh SIEM Alerting](secure_web_gateway.md)

Utilize Squid proxy, along with a OPNsense firewall, to block malware before it is downloaded, and log an event inside of a Wazuh SIEM.

---

### [HTML Smuggling Emulation & EDR Detection Via ELK Stack](ELK_Stack_&_Elastic_Defender_EDR.md)

This lab is a practical detection engineering and incident response simulation. It demonstrates how adversaries use HTML smuggling via a phishing lure to bypass network perimeters and deliver a stageless Meterpreter payload. Defensively, it showcases how an active EDR (Elastic Defend) intercepts the payload assembly on a Windows 11 endpoint, followed by incident triage, alert aggregation into case management, and root-cause analysis using the ELK Stack SIEM.

---

### [Suricata IPS Log Pipeline Into ELK Stack](Ingest_Suricata_Logs_Into_ELK.md)

Stand up Suricata IPS/IDS on an OPNsense firewall, and ingest Suricata alerts and log data into an ELK stack SIEM. This allows me to correlate IPS alerts with any Elastic EDR telemetry, to enrichen an attack story and provide me more data during investigations.
